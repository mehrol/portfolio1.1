How to use:
- unzip the folder and open index.html in a browser (internet connection required for CDN scripts: tsparticles and AOS).
- Replace assets/profile.png with your final image if needed.
- Replace CV.pdf with your real CV file.

Notes:
- This is a static HTML/CSS/JS portfolio built to match the requested dark-teal animated theme.
- For GitHub Pages, push the folder to a repo and enable GitHub Pages from the repo settings (use main branch / root).
